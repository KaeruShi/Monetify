# Supported Apps

Monetify currently supports the following apps with Monet theming and icon pack integration:

| App             | Support          |
| --------------- | ---------------- |
| DocumentsUI     | Monet, Icon Pack |
| GitHub          | Monet, Icon Pack |
| Instagram       | Monet            |
| Pinterest       | Icon Pack        |
| Pixiv           | _Soon_           |
| Reddit          | Monet, Icon Pack |
| Substratum Lite | Monet, Icon Pack |
| WhatsApp        | _Soon_           |
| X / Twitter     | Monet, Icon Pack |
| YouTube         | Monet            |

## Requirements

- Android 12+ (API 31+)
- Xposed Framework
- Root access

## Notes

Some apps may have partial support or behavior that depends on app updates. If a target app receives a major UI update, Monetify may require additional compatibility adjustments.
